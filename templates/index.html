Definitions
Word Error Rate (WER)
$$ WER = \frac{insertions + deletions + substitutions}{\text{ total number of words }} $$

Edit Distance
The edit distance is the minimum number of word edits (word insertions, deletions and substitutions) required to transform one value to another.
